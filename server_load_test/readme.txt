run locust using the following command:
    locust -f locustfile.py

Open your browser and navigate to http://localhost:8089.

Configure and Start the Test:

    - Number of Users: Set the total number of users to simulate.
    - Spawn Rate: Set the rate at which new users are spawned.
    - Host: Set the host URL as http://localhost:5223 (or the appropriate host if different).
    - Click the "Start Swarming" button to begin the test.
